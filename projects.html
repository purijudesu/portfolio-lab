Fatigue AI

Machine Learning

A Vision-Based Model for Fatigue Detection and Classification

As System Architect and ML Developer, I developed an advanced computer vision system that detects and classifies human fatigue through comprehensive analysis of eye blink rate, head pose estimation, and upper body posture.

The system uses sophisticated algorithms including Eye Aspect Ratio (EAR) and PERCLOS (Percentage of Eyelid Closure) to accurately assess fatigue levels in real-time. I designed and implemented a custom Multimodal Routing Algorithm that combines multiple input streams for enhanced classification accuracy.

Tools & Technologies Used:

  • Python
  • YOLOv8
  • MediaPipe
  • OpenCV
  • Computer Vision
  • Machine Learning
  • Algorithm Design

Key Achievements:

  • Trained and optimized multiple computer vision models
  • Implemented EAR & PERCLOS algorithms for accurate fatigue detection
  • Designed Multimodal Routing Algorithm for classification
  • Real-time processing and analysis capabilities
